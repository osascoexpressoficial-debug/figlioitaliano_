# Proposta Figlio Italiano × Osasco Express - Versao Final GitHub

## Publicacao
Suba estes arquivos na raiz do repositorio GitHub Pages:

- `index.html`
- `assets/`
- `contratos/`
- `site.webmanifest`

## O que foi feito
- Layout baseado no modelo mestre comercial aprovado pelo usuario.
- Identidade visual trocada para Figlio Italiano: verde escuro, creme, dourado e vermelho italiano.
- Texto ajustado para novo contrato e implantacao, sem linguagem de cliente antigo.
- Valores comerciais mantidos no modelo mestre: planos, janelas, reforco, km ate 10 km e adicional climatico.
- Contrato PDF proprio da Figlio Italiano anexado na pasta `contratos/`.
- Revisao para nao deixar sobras de cliente anterior no site ou no contrato.

## Arquivo principal
Abra `index.html` no navegador ou publique direto no GitHub Pages.
